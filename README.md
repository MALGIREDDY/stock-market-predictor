#  Stock Market Trend Predictor

This project predicts whether a stock will go **up or down** the next day based on historical data.

###  Tech Stack:
- Python
- Flask
- Machine Learning (Random Forest)
- yFinance API

###  Features:
- Fetches real stock data from Yahoo Finance
- Trains ML model to predict next-day trend
- Displays prediction & model accuracy
- Simple Flask web UI

###  Example Symbols:
- AAPL → Apple  
- TSLA → Tesla  
- MSFT → Microsoft  
- INFY.NS → Infosys India  

###  Run the project:
```bash
python app.py
